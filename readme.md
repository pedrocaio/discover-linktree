<h1 align="center"> LinkTree </h1>

<br>

## 💻 Projeto

O LinkTree é um agregador de links, feito para estudo e para facilitação no acesso a algumas aplicações.

<br>

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma
